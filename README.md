# Data Science — From Scientific Method to Modern AI

Repository created during the workshop **“Data Science — From Scientific Method to Modern AI”**, presented by **Renan Lafayete Biagio** from ROQT Data & AI.

The workshop explores how the scientific method connects with modern Data Science and Artificial Intelligence practices, demonstrating how analytical foundations can be transformed into real-world solutions using Python and VS Code.

## About the Workshop

This workshop covers:

- Fundamentals of Data Science
- Scientific thinking applied to data analysis
- Introduction to Artificial Intelligence concepts
- Practical examples using Python
- Development workflow with VS Code
- Real-world applications and industry insights
